Sobre mim

Olá, meu nome é Pedro e sou de Ribeirão Preto, São Paulo. Sou desenvolvedor de software com quatro anos de formação e três anos de experiência profissional na área. Minha paixão pela tecnologia surgiu quando me matriculei no curso de ADS na universidade, onde criei meu primeiro programa usando a linguagem de programação C - um jogo de hash (jogo da velha) que me apresentou ao mundo da programação e do desenvolvimento de software. Desde então, sou movido por uma sede de conhecimento e busquei cursos extracurriculares em áreas como Desenvolvimento Orientado a Objetos, Desenvolvimento Orientado a Testes e Inteligência Artificial.

Minha experiência mais recente de trabalho foi em uma cooperativa financeira, onde trabalhei no desenvolvimento de software para suas operações. Nesse papel, usei Angular, Angular Material, SCSS e Tailwind para o front-end e C #, ASP.NET Core, DDD e CQRS para o back-end. Também adquiri experiência com testes unitários e testes automatizados. Antes disso, trabalhei em uma empresa de desenvolvimento de software onde contribuí para vários projetos para clientes em diferentes setores. Um desses clientes foi uma corretora de valores, onde trabalhei em uma plataforma de negociação. Esse projeto envolveu o uso de tecnologias como C #, .NET Core, Entity Framework Core, DDD e TDD. Também adquiri experiência com tecnologias front-end como Angular, SCSS e Bootstrap.

Minhas áreas de interesse em desenvolvimento de software incluem Desenvolvimento Orientado a Objetos, Desenvolvimento Orientado a Testes, Programação Reativa e Arquitetura Orientada a Eventos. Tenho experiência em uma variedade de tecnologias, incluindo linguagens de programação como C# (Framework/Core), JavaScript e TypeScript. Também estou familiarizado com estruturas de teste como xUnit e Jasmine. Além disso, trabalhei com bancos de dados como Oracle, PostgreSQL, MySQL, MongoDB e MSSQL e sou proficiente em criar diagramas UML usando o yEd.

Obrigado por considerar meu perfil. Estou confiante de que minha experiência e habilidades seriam uma contribuição valiosa para sua organização. Por favor, sinta-se à vontade para entrar em contato comigo se tiver alguma dúvida adicional ou se desejar agendar uma entrevista.
